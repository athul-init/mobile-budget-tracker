# 📱 FinTrack Mobile

A lightweight, mobile-first budget tracking web application designed for seamless personal expense monitoring. Built with clean, semantic modern web standards and responsive view ceilings.

## ✨ Features
* **Mobile-Optimized Interface:** Fixed viewport layout scaling meticulously modeled for smooth mobile interaction.
* **Persistent Local Storage:** Transactions remain saved locally in the browser cache, preventing data loss between sessions.
* **Real-Time Data Aggregation:** Automatically updates current balance and computes asset allocations dynamically.
* **Chart.js Integration:** Renders an interactive doughnut chart breaking down expense distributions natively.

## 🛠️ Tech Stack
* **Markup/Layout:** HTML5, CSS3 Custom Properties
* **Logic Engine:** Vanilla JavaScript (ES6+ Array Methods like `.reduce()` and `.filter()`)
* **Visualization:** Chart.js Library via CDN

## 📈 Engineering Highlights
* Single-file state architecture built entirely inside **Acode Editor** on a mobile environment.
* Optimized event-handling logic modifying chart parameters in-place to avoid layout memory leaks.
